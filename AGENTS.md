## プロジェクト概要

- `nth_weekday` は「第3水曜」「最終金曜」などの日時を計算する、小規模かつ依存の少ない Ruby gem です。
- 対応 Ruby バージョン: 3.3.0 以上。

本ファイルは、このリポジトリに対して作業するエージェント（Codex 等）向けの最上位ガイドです。

---

## 優先順位（矛盾時の解決順）

ルールが矛盾する場合は、以下の優先順位で解釈します。

1. `AGENTS.md`
2. `HUMAN_IN_THE_LOOP.md`
3. `CODING_RULES.md`
4. `README.md` やコードコメントなどの参考情報

---

## 動作モード

このリポジトリには2つの作業モードがあります。

### 1. 通常モード（デフォルト）

単一エージェントによる実装・修正を行う。以下の条件の場合はこちらを使う：

- 小さな修正
- 単純なバグ対応
- 明確なタスク

---

### 2. オーケストレーターモード

以下の場合は `.codex/AGENTS.md` を参照し、multi-agent フローで処理する：

- 要件が曖昧
- 複数ステップ（設計・実装・レビュー）が必要
- 大きな変更
- ユーザーが「orchestrate」「オーケストレーター」等を明示した場合

👉 この場合は必ず `.codex/AGENTS.md` に従うこと

---

## 基本方針（必ず守ること）

- 原則として **新しい runtime 依存は追加しない**（標準ライブラリ優先）。
- Ruby バージョン要件を変更する場合は、理由を明記し CI も更新する。
- 公開 API は後方互換を優先する。
- 不要なリファクタリングやフォーマット変更を混ぜない。
- 小さく、目的が明確な差分を心がける。
